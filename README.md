# Assessment-AI-Assisted-Programming
This repository contains the practical implementation for Assessment 2 of the Advanced Programming Concepts module. The objective of this project is to develop a software solution using two distinct development methodologies to critically evaluate the technical and ethical implications of Artificial Intelligence in software engineering

The system allows users to:
- Add books
- Add members
- Display all books
- Display all members
- Borrow books
- Return books
- Search books by title
- Search books by author
- Search books by category

This repository includes both:
- Version A – Manual implementation (without AI assistance)
- Version B – AI-assisted implementation

## Technologies Used
- Java
- IntelliJ IDEA
- Git & GitHub

## How to Run
1. Clone this repository.
2. Open the project in IntelliJ IDEA.
3. Compile the project.
4. Run either:
   - VersionA (Manual Approach)
   - VersionB (AI Approach)
